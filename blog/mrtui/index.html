Fetching Bitcoin Mempool Data from Terminal

DATE

For those of us who are doing most of their work in the terminal, it can be very annoying to switch to a web browser for small tasks. For instance, looking something up like the current block height in the Bitcoin network or the current network fees. Granted, most people probably don't mind, but I do! So I wrote a little CLI tool to fetch some data from mempool.space and display the info in the terminal.

The output is pretty self-explanatory and is kept as similar to the mempool.space layout as possible. However, it is static data, meaning we only get the receive data once. If you are looking for a live data feed (like it is on the mempool.space website) then check out mempool-cli.

The program is written in Rust, in case you don't have Rust by now, simply install it by running the rustup command. It's literally just a one command to install the whole language. Once rust is all set up, you can install Mr. TUI (Mempool Rust Terminal UI) by simply running this command. 

cargo install mrtui

Then, whenever you want to output the current blockchain data, run the following. 

mrtui

That's it! Now you can get simple blockchain information about the bitcoin network through the terminal.
